Interface Prototype #2

Link to Figma!

This week, I tried thinking of some new ways to layout my navigation and site as a whole. I made my h1-h4 hierarchy and attempted to decide how that would look on the site. I made my landing page simple, as I didn't want to go too crazy and have something unachievable, especially with the limited time we have. I came up with how the user would find the different pages and I believe I made it much easier. There will be one page navigating to sketches 1-10 (ten being the micro project) and one for the other pages, for example, one like this one. This would also include the reading responses, progress checks from the micro project, etc. I made the sample page for the sketches and I used stars to make it more interesting. In addition, this allows me to add images for each sketch inside the stars as well. The final page I did was the micro project page. For this I added where the sketch would be along with the explanation section to really fill out the space. Since the sketch is more so sized for desktop, thinking of how to make it look normal on mobile was interesting.

For the future, I'd like to get some feedback first, but I think considering some of the other layouts I have on my site in this new format will be the challenge. Especially all of the text. I think that's one of the main things I'd like to address. The reading responses and progress checks need something to make them look cleaner as right now they are just on the background. I also would like to think about transferring some of the other layouts and make the whole site interesting and usable. Slightly irrelevant for now, but I'd also like to find a new body font as the one I have currently is a bit illegible.
